# Validador de Credenciales con Expresiones Regulares 

Este proyecto es una aplicación de escritorio desarrollada en **Java** que implementa la validación estricta de formatos de correo electrónico y robustez de contraseñas utilizando **Expresiones Regulares (Regex)** y fundamentos de la Teoría de Autómatas.

##  Características
* **Validación de Correo:** Comprobación de estructuras válidas (usuario, dominio y extensión).
* **Validación de Contraseña:** Filtrado por políticas de seguridad (mayúsculas, minúsculas, números y caracteres especiales).
* **Base Teórica:** Modelado previo de las expresiones mediante Autómatas Finitos Deterministas (AFD).
